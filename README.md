# SoSI - Spirit of Security Information

Веб-платформа для услуг информационной безопасности. Включает пентестинг, анализ кода, консалтинг и техподдержку. Современный адаптивный интерфейс с системой оплаты и чатом с исполнителями.

## Структура проекта

- `index.html` - главная страница
- `html/` - дополнительные страницы
- `css/` - стили
- `js/` - JavaScript код
- `favicon.ico` - иконка сайта
